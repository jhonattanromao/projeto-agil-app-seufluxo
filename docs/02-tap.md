# 02 – Termo de Abertura do Projeto (TAP)

O TAP é um documento essencial que formaliza escopo, objetivos, cronograma, orçamento e riscos iniciais.  

O documento foi desenvolvido antes de iniciar a organização das
tarefas e funcionalidades dentro do Asana. É de extrema importância entender as necessidades do cliente, os riscos do projeto, a duração, orçamento necessário e etc, por isso é imprescindível a criação deste documento.
