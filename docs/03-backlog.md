# 03 – Product Backlog

O Backlog do Produto foi desenvolvido com as principais telas do aplicativo proposto, conforme estrutura abaixo:

`- Interface Front-End:`  
Todo o desenho do aplicativo estruturado em código;  

`- Página de Login:`  
Página de autenticação do usuário;  

`- Página Inicial:`  
Tela de início ao acessar a conta;  

`- Adição de Lançamentos:`  
Página de criação de lançamentos (receita/despesa);  

`- Configurações do App:`  
Configurações gerais do aplicativo;  

`- Configurações do Usuário:`  
Configurações da conta do usuário;
